# TechWriter Portfolio

A modern, responsive portfolio website built with Next.js and Tailwind CSS. Features include:

- Dark/light theme toggle
- Responsive design
- Motion effects and animations
- Project showcase
- TechWrites blog section
- Contact form

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

This site is automatically deployed to GitHub Pages using GitHub Actions.

Visit the live site: [https://heyadrsh.github.io/tech-portfolio/](https://heyadrsh.github.io/tech-portfolio/)

## Technologies Used

- Next.js
- React
- Tailwind CSS
- TypeScript
- Intersection Observer API for animations
